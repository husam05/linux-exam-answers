Question 1: Practical Command Execution (10 points)

Command: whoami

Expected sample output:

student

Explanation notes:

  • Prints the effective username executing the shell session.
  • Useful in scripts and after using sudo to confirm who is running commands.

Question 2: Theoretical Knowledge (10 points)

MCQ: Which Linux distribution is purpose-built for cybersecurity professionals and penetration testing?

Correct answer: b) Kali Linux

Justification: Kali Linux ships with an extensive suite of offensive security and penetration-testing tools out of the box.
